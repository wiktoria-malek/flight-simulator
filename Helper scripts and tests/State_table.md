Conclusions:

Everything that changes settings of the machine needs to be in the Interface. State stores a snapshot of the machine.
State saves standarized format and allows to load it. Without it, we'd have to have a different saving format for each interface.
So, State allows us to read and save data from the machine.
It also filters names, avergaes orbit and stores data.
